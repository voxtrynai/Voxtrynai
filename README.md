# VoxtrynAI 🚀

## World's First All-in-One AI Super App

![VoxtrynAI Banner](https://voxtrynai.base44.app/banner.png)

> **All AI. One Place. Forever FREE.** 🎯

VoxtrynAI is the world's first all-in-one AI super app combining multiple AI models, Memory Palace, Emotion Engine, and Agent Swarm in one completely free platform.

## ✨ Key Features

### 🧠 Memory Palace
Persistent AI memory system that remembers your preferences and past conversations for personalized experiences.

### ❤️ Emotion Engine
Emotionally intelligent AI responses that understand and adapt to your emotional context.

### ⚡ Agent Swarm
Multiple AI agents working together intelligently to solve complex problems and tasks.

### 🎨 Image Generation
AI-powered image creation with advanced customization options.

### 🎙️ Voice Chat
Natural voice conversations with AI in multiple languages.

### 🗺️ Mind Map
Visual AI-powered idea mapping for brainstorming and organization.

### 🤖 Multiple AI Models
Access to various AI models (GPT, Claude, Llama, etc.) all in one place via OpenRouter API.

### 📊 Analysis Studio
Deep AI-powered analysis for data, content, and insights.

### 🎬 Video Maker
AI video creation tool for content creators.

### 🌍 Live World Brain
Real-time world knowledge and current events integration.

## 📦 Installation & Setup

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Git

### Clone the Repository
```bash
git clone https://github.com/voxtrynai/Voxtrynai.git
cd Voxtrynai
```

### Install Dependencies
```bash
npm install
# or
yarn install
```

### Environment Setup
Create a `.env.local` file in the root directory:
```env
REACT_APP_API_URL=your_api_url
REACT_APP_OPENROUTER_KEY=your_openrouter_key
REACT_APP_BASE44_API_KEY=your_base44_key
```

See [.env.example](./.env.example) for all available options.

### Run Development Server
```bash
npm start
# or
yarn start
```

The application will open at `http://localhost:3000`

### Build for Production
```bash
npm run build
# or
yarn build
```
